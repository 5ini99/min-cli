# min stylus 编译器

## 安装

``` bash
$ npm install @mindev/min-compiler-stylus --save-dev
```


## 配置 `min.config.js`

``` js
module.exports = {
  compilers: {
    stylus: {
      compress: true
    }
  }
}
```
